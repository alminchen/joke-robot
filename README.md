# joke
> A Vue.js project
本地调试启动：

第零步：代码clone到本地： git clone https://github.com/alminchen/joke-robot.git

第一步：npm install

 a:或者用淘宝的cnpm代理安装会更快，方法是：
 1.安装cnpm：npm install -g cnpm --registry=https://registry.npm.taobao.org 
 2.用cnpm代理npm安装： cnpm install

第二步：npm run dev
打包：npm run build
